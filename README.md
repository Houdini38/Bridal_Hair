# Angela Bernard Hair + Brooklyn Beauty

A responsive marketing website for an on-location bridal &amp; special event
hair styling team serving Denver, Broomfield, and surrounding Colorado venues.

## Sections

- **Hero** — headline, call-to-action buttons, and a "Request Event Availability" inquiry form
- **Feature bar** — On-Location Styling · Bridal Parties · Timeline Planning · Colorado Travel
- **Meet Your Stylists** — Angela Bernard &amp; Brooklyn Martinez
- **Services** — Wedding Day Hair, Bridal Party Styling, Preview Trials, Special Event Styling, Travel Services
- **How It Works** — Inquire → Confirm Details → Plan Your Look → Event Day Styling
- **Event Moments** — photo gallery
- **Love Notes** — client testimonials
- **FAQ** — accordion of common questions
- **Final CTA** &amp; footer

## Tech

Plain HTML, CSS, and vanilla JavaScript — no build step required.

| File | Purpose |
| --- | --- |
| `index.html` | Page markup and content |
| `styles.css` | Styling, layout, and responsive breakpoints |
| `script.js` | Mobile nav, inquiry form handling, testimonial dots |

## Running locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
